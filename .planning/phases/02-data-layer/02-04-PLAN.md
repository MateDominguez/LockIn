    ```python
        "prices": 0,
        "fundamentals": 7,
        "macro": 14,
    }
    ```

    **Constructor:**
    - `__init__(self, source: DataSourceProtocol, macro: MacroSourceProtocol)` — accepts protocol-typed sources so any implementation can be injected.

    **Method: `get_fundamentals(self, ticker: str, as_of_date: date | None = None) -> FundamentalsResult`**
    1. **Future date guard:** If `as_of_date` is not None and `as_of_date > date.today()`, raise `ValueError(f"as_of_date {as_of_date} is in the future (today={date.today()})")`.
    2. **Live analysis bypass:** If `as_of_date` is None or `as_of_date == date.today()`, delegate directly to `self._source.get_fundamentals(ticker, as_of_date=None)` — no date enforcement.
    3. **Historical analysis:** Delegate to `self._source.get_fundamentals(ticker, as_of_date=as_of_date)` — the source applies the 7-day lookahead tolerance internally.
    4. Return the result unchanged (validation and storage happen in the public API layer, not here).

    **Method: `get_macro_indicators(self, as_of_date: date | None = None) -> MacroResult`**
    1. **Future date guard:** Same as above.
    2. **Live analysis bypass:** If `as_of_date` is None or `as_of_date == date.today()`, delegate to `self._macro.get_macro_indicators(as_of_date=None)`.
    3. **Historical analysis:** Delegate to `self._macro.get_macro_indicators(as_of_date=as_of_date)` — FREDSource uses `get_series_as_of_date()` internally for ALFRED vintage access.
    4. Return the result unchanged.

    The PointInTimeData class is a thin wrapper that enforces the date contract. It does NOT duplicate the date filtering logic already in the sources — it only handles the future date guard and the live-analysis bypass.

  <files>src/lockin/data/__init__.py</files>
  <action>
    Replace the empty `src/lockin/data/__init__.py` with the public API module.

    **Module docstring:** Explain that this is the public API for the data layer. Agents import from here, never from submodules directly.

    **Module-level lazy initialization:**

    Create a module-level `_default_pit: PointInTimeData | None = None` and a `_default_validator: DataValidator | None = None`.

    Helper `_get_pit() -> PointInTimeData`: If `_default_pit` is None, construct it with `YFinanceSource()` and `FREDSource()`. Cache in module global. Return it.

    Helper `_get_validator() -> DataValidator`: If `_default_validator` is None, construct with `get_settings().database_url`. Cache. Return.

    **Public function: `get_fundamentals(ticker: str, as_of_date: date | None = None, store: bool = True) -> FundamentalsResult`**
    1. Call `_get_pit().get_fundamentals(ticker, as_of_date)` to get raw data.
    2. Call `_get_validator().validate_fundamentals(result)` to get ValidationResult.
    3. Merge validation into the FundamentalsResult: set `result["missing_fields"]` from validation, set `result["outlier_flags"]` from validation.
    4. **Lazy storage:** If `store` is True and `get_settings().database_url` is not empty:
       - Extract fiscal_year_end from `result.get("fiscal_year_end")` — YFinanceSource now populates this from the DataFrame column date (Plan 02-02). Fallback: `as_of_date or date.today()` if fiscal_year_end is None.
       - Also call `store_asset(database_url, ticker)` to upsert the ticker in the assets table (lazy: just the ticker, no metadata enrichment here).
       - Call `store_fundamentals(database_url, ticker, result, fiscal_year_end)`.
       - Wrap in try/except — storage failure should NOT break the data fetch. Log error to stderr.
    5. Return the FundamentalsResult.

    **Public function: `get_macro_indicators(as_of_date: date | None = None, store: bool = True) -> MacroResult`**
    1. Call `_get_pit().get_macro_indicators(as_of_date)` to get raw data.
    2. **Lazy storage:** If `store` is True and `get_settings().database_url` is not empty:
       - Call `store_macro_data(database_url, result)`.
       - Wrap in try/except — storage failure should NOT break the data fetch.
    3. Return the MacroResult.

    **Exports (`__all__`):**
