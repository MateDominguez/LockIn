  <action>
    Create `src/lockin/data/yfinance_source.py` with class `YFinanceSource`:

    **Constructor:**
    - `__init__(self, cache: TTLCache | None = None)` — accepts optional cache injection (for testing). If None, create a new TTLCache instance.

    **Private helper: `_fetch_ticker`**
    - Decorated with `@retry(retry=retry_if_exception_type(Exception), stop=stop_after_attempt(3), wait=wait_exponential(multiplier=1, min=2, max=30))` from tenacity.
    - Calls `yf.Ticker(ticker_sym)` and returns the Ticker object.
    - This is the retry boundary — rate limit errors (HTTP 429) trigger backoff here.

    **Public method: `get_fundamentals(self, ticker: str, as_of_date: date | None = None) -> FundamentalsResult`**

    Logic flow:
    1. Build cache key: `f"fundamentals:{ticker}"` (date not in key — we cache the raw DataFrames and filter after)
    2. Check cache: `self._cache.get(key, TTL_FUNDAMENTALS)`. If hit, use cached raw data.
    3. On cache miss: call `_fetch_ticker(ticker)`, then read `ticker.income_stmt`, `ticker.balance_sheet`, `ticker.cashflow`.
       - CRITICAL: Use `income_stmt` NOT `financials` (deprecated, returns empty).
       - If ALL three DataFrames are empty, try `self._cache.get_stale(key)` for stale fallback.
       - If still no data, `raise DataUnavailableError(ticker=ticker, source="yfinance")`.
    4. Cache the raw DataFrames dict on successful fetch: `self._cache.set(key, raw_data)`.
    5. Point-in-time filtering: If `as_of_date` is not None and `as_of_date != date.today()`:
       - Compute `cutoff = as_of_date + timedelta(days=7)` (7-day lookahead tolerance for fundamentals).
       - Filter each DataFrame's columns: keep only columns where `pd.Timestamp(col).date() <= cutoff`.
       - Filter by DATE not by position index (avoid look-ahead bias).
    6. Extract fields from the most recent valid column (index 0 after filtering).
       - Use `df.loc[field_name]` with a dict mapping: `{"TotalRevenue": "total_revenue", "NetIncome": "net_income", "GrossProfit": "gross_profit", "OperatingIncome": "operating_income", "EBITDA": "ebitda", "DilutedEPS": "diluted_eps", "FreeCashFlow": "free_cash_flow", "TotalAssets": "total_assets", "TotalDebt": "total_debt", "CashAndCashEquivalents": "cash_and_equivalents", "StockholdersEquity": "total_equity"}`.
       - For each field, use `.get()` pattern or try/except KeyError — if field not in DataFrame index, record in `missing_fields`.
       - Convert numpy/pandas values to Python float via `float(val)` if not NaN, else None.
    7. **fiscal_year_end extraction:** Take the most recent valid column date used (i.e. the first column after filtering, which is `income_stmt.columns[0]` after filtering). Convert to `date` via `pd.Timestamp(col).date()`. Set as `fiscal_year_end` in the result. If the DataFrame has no valid columns (all filtered out), set `fiscal_year_end = None`.
    8. Build and return `FundamentalsResult` with all extracted fields, fiscal_year_end (from step 7), source="yfinance", fetched_at=now(UTC), as_of_date=(ISO string or "live"), missing_fields list, outlier_flags={} (empty — validator fills this later), data_freshness="FRESH" (or "STALE" if from stale cache).

    **Error handling:**
    - Wrap the entire fetch in try/except. On any exception after retry exhaustion:
      - Try stale cache fallback first.
      - If no stale data exists, raise `DataUnavailableError(ticker=ticker, source="yfinance")`.
    - Partial data (some fields None) is OK — return the result with `missing_fields` populated.

    **Important:** Do NOT use `ticker.financials` or `ticker.earnings` — both are deprecated and return empty DataFrames silently.
  </action>

    Create `src/lockin/data/fred_source.py` with class `FREDSource`:

    **Module-level constant** `FRED_SERIES` dict mapping indicator names to FRED series IDs:
    ```python
    FRED_SERIES = {
        "gdp": "GDP",
        "cpi": "CPIAUCSL",
        "core_pce": "PCEPILFE",
        "fed_funds": "FEDFUNDS",
        "yield_10y_2y": "T10Y2Y",
        "yield_10y_3m": "T10Y3M",
        "unemployment": "UNRATE",
        # ISM Manufacturing PMI (NAPM = historical NAPM/ISM series on FRED)
        # If NAPM is unavailable, fall back to BSCICP02USM460S (OECD manufacturing confidence)
        "manufacturing_pmi": "NAPM",
    }
    ```
    Note: If the NAPM series fetch returns an empty result at runtime, catch the error, log a warning to stderr, and set `manufacturing_pmi = None` — the partial-data fallback handles this gracefully. Do not block the entire macro fetch over a single uncertain series ID.

    **Constructor:**
    - `__init__(self, api_key: str | None = None, cache: TTLCache | None = None)`:
      - If `api_key` is None, read from `get_settings().fred_api_key`.
      - If still empty, raise `DataUnavailableError(ticker="FRED", source="fred")` with a message about missing API key.
      - Create `self._fred = Fred(api_key=api_key)`.
      - Cache: if None, create new TTLCache instance.

    **Public method: `get_macro_indicators(self, as_of_date: date | None = None) -> MacroResult`**

    Logic flow:
    1. Build cache key: `f"macro:{as_of_date or 'live'}"`.
    2. Check cache: `self._cache.get(key, TTL_MACRO)`. If hit, return cached MacroResult.
    3. On cache miss, fetch each series:
       - For live (as_of_date is None or as_of_date == date.today()):
         Use `self._fred.get_series(series_id)`. Take `float(series.dropna().iloc[-1])` for the latest value.
       - For historical (as_of_date in past):
         Use `self._fred.get_series_as_of_date(series_id, str(as_of_date))`. This uses FRED's ALFRED vintage dates — returns data as known on that date (no revision look-ahead).
         Take `float(df["value"].iloc[-1])` if df is not None and not empty, else None.
       - Wrap each series fetch in try/except — if one series fails, set that indicator to None and continue (partial data is acceptable).
    4. Cache the result: `self._cache.set(key, result)`.
    5. If ALL indicators are None (total failure):
       - Try stale cache: `self._cache.get_stale(key)`.
       - If no stale data, raise `DataUnavailableError(ticker="FRED", source="fred")`.
    6. Build and return MacroResult with all indicators, source="fred", fetched_at=now(UTC), as_of_date=(ISO string or "live"), data_freshness="FRESH" (or "STALE" if stale fallback).

    **Error handling:**
    - Individual series failures: log to stderr, set value to None, continue.
    - Total failure: raise DataUnavailableError after trying stale cache.
    - Do NOT silently return None — always return MacroResult or raise.

    **Important:** Use `get_series_as_of_date()` for backtesting paths (prevents revision look-ahead per research). Only use `get_series()` for live analysis.
  </action>
