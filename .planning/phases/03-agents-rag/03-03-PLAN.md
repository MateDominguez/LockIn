  ```python
  def calculate_epv(ebit_5y_avg: float, tax_rate: float, wacc: float, shares_outstanding: float) -> float:
      if wacc <= 0: raise ValueError("WACC must be positive")
      after_tax = ebit_5y_avg * (1 - tax_rate)
      return after_tax / wacc / shares_outstanding

  def calculate_eva(nopat: float, wacc: float, invested_capital: float) -> float:
      return nopat - (wacc * invested_capital)

  def calculate_rim(book_value: float, roe: float, cost_of_equity: float, growth_rate: float, shares_outstanding: float) -> float:
      if cost_of_equity <= growth_rate: raise ValueError("cost_of_equity must exceed growth_rate")

      total_value = book_value * (1 + residual_spread)
      return total_value / shares_outstanding
  ```

     - EPV: use operating_income as EBIT proxy, assume WACC=0.10, tax_rate from data or 0.21 default

  6. Build ValueDistribution (log-normal parametrized) per Notion spec:
     - Use the mean and spread of valuation model results to estimate log-normal parameters.
     - Compute mu and sigma for log-normal from the intrinsic value estimates:
       - If multiple models applicable, mu = log(mean of model values), sigma from spread
       - If single model, mu = log(intrinsic_value), sigma = 0.20 (default uncertainty)
     - Use scipy.stats.lognorm to compute P10, P50, P90 from mu/sigma:
       ```python
       import numpy as np
       from scipy.stats import lognorm
       mu = np.log(intrinsic_value)
       sigma = 0.20  # or computed from model spread
       dist = lognorm(s=sigma, scale=np.exp(mu))
       p10 = dist.ppf(0.10)
       p50 = dist.ppf(0.50)  # median = exp(mu)
       p90 = dist.ppf(0.90)
       ```

       ```python
       data_coverage = DataCoverage(

           missing=["segment_breakdown", "management_guidance", ...],
           confidence_impact=-0.10 * len(missing) / (len(available) + len(missing)),
       )
       ```
     - Compute confidence based on data completeness (0.0-1.0)

     ```python
     from lockin.agents.types import ValueDistribution, DataCoverage

     bull_distribution = ValueDistribution(
         expected_value=intrinsic_value,
         std_dev=intrinsic_value * sigma,  # in $ terms
         p10=p10,
         p50=p50,
         p90=p90,
         confidence=confidence,
     ```
