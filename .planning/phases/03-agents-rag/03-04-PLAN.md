Implement the Strategist agent outputting a ConfidenceModifier per the Notion Judge spec. VeTO adjusts variance only (no base rate), analyst momentum has a calibrated base rate.

Purpose: Strategist is a Modifier agent (Family 2). It does NOT opine on price -- it opines on how much to trust Distribution agents. VeTO is an organizational health indicator that increases uncertainty (variance) when unreliable, but does NOT adjust probability because it has no empirical validation. Analyst momentum has a calibrated base rate from Jegadeesh (2004).
Output: strategist.py outputting ConfidenceModifier with properly typed Signals.

  ```python
  def strategist(state: InvestmentState, config: RunnableConfig) -> dict:
  ```

  Implementation:
  1. Get ticker from state["asset_ticker"]

  2. Fetch earnings transcript from FMP API (if fmp_api_key configured):
     - URL: `https://financialmodelingprep.com/api/v3/earning_call_transcript/{ticker}?apikey={key}`
     - Use httpx.get() with 10s timeout
     - Cache response (simple module-level dict cache by ticker)
     - If FMP key not set or request fails, skip transcript analysis gracefully

  3. Fetch analyst consensus from yfinance:
     - `yf.Ticker(ticker).recommendations_summary` for recent analyst ratings
     - Extract: buy/hold/sell counts, target price consensus
     - Compute analyst_momentum: net upgrades vs downgrades
     - If unavailable, proceed without

  4. Build LLM prompt (MODEL_FLASH) for sentiment analysis:
     - System: "You are a financial strategist analyzing earnings calls and analyst consensus. Extract: 1) Overall sentiment (bullish/neutral/bearish, 0-1 score), 2) Key topics/keywords from transcript, 3) Management tone assessment, 4) VeTO score (0-1, organizational health indicator based on language patterns). Respond in JSON."
     - Human: Include transcript excerpt (first 3000 chars to stay within context), analyst summary, ticker info.

  5. Parse LLM response, extract sentiment score and VeTO score.

  6. Build ConfidenceModifier per Notion spec:
     ```python
     from lockin.agents.types import ConfidenceModifier, Signal, DataCoverage

     # Margin adjustments per spec:
     margin_adj = 0.0
     if veto_score < 0.4:  # VeTO low
         margin_adj += 0.05
     if analyst_momentum < 0:  # Net analyst downgrades
         margin_adj += 0.05

     # Variance adjustments per spec:
     # VeTO has NO base rate -> adjusts variance only
     variance_adj = 0.0
     if not veto_has_base_rate:  # Always True for VeTO -- no empirical validation
         variance_adj += 0.10  # More uncertainty because signal not calibrated

     signals = [
         Signal(
             name="veto_score",
             value=veto_score,
             category="low" if veto_score < 0.4 else "medium" if veto_score < 0.7 else "high",
             has_base_rate=False,    # CRITICAL: VeTO has no base rate
             base_rate=None,
             base_rate_source=None,
         ),
         Signal(
             name="analyst_momentum",
             value=analyst_momentum_value,
             category="upgrade" if analyst_momentum_value > 0 else "downgrade" if analyst_momentum_value < 0 else "neutral",
             has_base_rate=True,     # Has calibrated base rate
             base_rate=analyst_base_rate,  # From Jegadeesh (2004) placeholder
             base_rate_source="Jegadeesh (2004)",
         ),
         Signal(
             name="sentiment_price_impact",
             value=sentiment_score,
             category=sentiment_label,
             has_base_rate=False,
         ),
     ]

     strategist_modifier = ConfidenceModifier(
         margin_adjustment=margin_adj,
         variance_adjustment=variance_adj,
         circuit_breaker=False,          # Strategist NEVER triggers circuit breaker
         circuit_breaker_reason=None,
         signals=signals,
         data_coverage=DataCoverage(
             available=available_sources,
             missing=missing_sources,
             confidence_impact=confidence_impact,
         ),
         reasoning=narrative,
     )
     ```

  7. Return InvestmentState-compatible dict:
     ```python
     return {
         "strategist_modifier": strategist_modifier,  # ConfidenceModifier instance
         "strategist_veto": veto_score,       # 0-1, kept for backward compat
         "strategist_sentiment": sentiment,    # 0-1
         "strategic_signals": signals_dict,    # Legacy dict format
         "strategist_narrative": narrative,
         "strategist_confidence": confidence,
     }
     ```

  Note: Add `strategist_modifier: ConfidenceModifier | dict` to InvestmentState TypedDict.
