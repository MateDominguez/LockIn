  <action>
  Create src/lockin/agents/judge.py:

  ```python
  def judge(state: InvestmentState, config: RunnableConfig) -> dict:
  ```

  Implementation:
  1. Read Bull and Bear distributions from state:
     - bull_dist = state.get("bull_valuation_distribution", {})
     - bear_dist = state.get("bear_valuation_distribution", {})
     - bull_conviction = state.get("bull_confidence", 0.5)
     - bear_conviction = state.get("bear_conviction", 0.5)

  2. Bayesian synthesis (from research finding #14):
     ```python
     # Weighted posterior
     total_weight = bull_conviction + bear_conviction
     posterior_mean = (bull_conviction * bull_dist["mean"] + bear_conviction * bear_dist["mean"]) / total_weight
     # Variance combines weighted variances + between-group variance
     bull_var = bull_dist.get("std_dev", 20) ** 2
     bear_var = bear_dist.get("std_dev", 20) ** 2
     weighted_var = (bull_conviction * bull_var + bear_conviction * bear_var) / total_weight
     between_var = bull_conviction * bear_conviction * (bull_dist["mean"] - bear_dist["mean"])**2 / total_weight**2
     posterior_std = math.sqrt(weighted_var + between_var)
     ```

  3. Generate consensus distribution percentiles using scipy.stats.norm.

  4. Compute conviction and divergence:
     - conviction = 1 - (posterior_std / posterior_mean) if posterior_mean > 0 else 0
     - divergence = abs(bull_dist["mean"] - bear_dist["mean"]) / max(bull_dist["mean"], bear_dist["mean"])

  5. Determine recommendation:
     - Get current price from state (or fetch via yfinance)
     - margin_of_safety = (posterior_mean - current_price) / posterior_mean
     - BUY if margin > guardian_adjusted_margin (from state)
     - SELL if margin < -0.10 (overvalued by 10%+)
     - HOLD otherwise

  6. HITL triggers (from research finding #14):
     - Trigger if conviction < 0.5
     - Trigger if divergence > 0.30
     - Set judge_hitl=True, judge_hitl_reason with details

  7. Retrieve RAG citations (if available):
     - query = f"{ticker} valuation intrinsic value"
     - citations = retrieve_with_citations(query, k=3)
     - Add to state as citations list

  8. Invoke LLM (MODEL_PRO) for narrative synthesis:
     - System: "You are a judicial investment analyst synthesizing Bull and Bear arguments. Provide: 1) Your recommendation (BUY/HOLD/SELL), 2) Key reasoning, 3) What would change your mind (risk factors), 4) Confidence level. Be balanced and reference both sides."
     - Human: Include Bull thesis, Bear thesis, Bayesian synthesis results, guardian risk report, strategist signals.

  9. Return:
     ```python
     return {
         "judge_consensus_distribution": consensus_dict,
         "judge_recommendation": recommendation,
         "judge_conviction": conviction,
         "judge_margin": margin_of_safety,
         "judge_price_target": posterior_mean,
         "judge_narrative": narrative,
         "judge_hitl": hitl_triggered,
         "judge_hitl_reason": hitl_reason,
         "citations": citations_list,
     }
     ```

  Note: The judge function replaces mock_judge. The judge_with_hitl wrapper in builder.py will call this function instead of mock_judge when graph wiring is updated (Plan 03-10).
