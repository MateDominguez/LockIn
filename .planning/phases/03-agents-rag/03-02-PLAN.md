<objective>
Implement the Macro Oracle agent that analyzes FRED macroeconomic data to detect economic regime and risk appetite, outputting a ConfidenceModifier per the Notion Judge spec.

Purpose: Macro Oracle is a Modifier agent (Family 2). It does NOT opine on price -- it opines on how much to trust Distribution agents. It produces signals with calibrated base rates (macro_base_rate from FRED historical data) that feed the Judge's probability calculation. It also establishes the macro context (expansion/contraction, risk-on/off) for downstream agents.
Output: macro_oracle.py with real LLM-powered regime detection producing ConfidenceModifier, unit tests with mocked LLM.

     # extreme_greed (low VIX + expansion): margin_adjustment +0.15 to +0.20

             category=flow_label,   # sector-level flow direction
             has_base_rate=False,
