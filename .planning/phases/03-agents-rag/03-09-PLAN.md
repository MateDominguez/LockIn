  <action>
  Create src/lockin/agents/optimizer.py:

  ```python
  # Constants
  MAX_POSITION_SIZE = 0.12     # 12% max per asset
  MAX_SECTOR_ALLOCATION = 0.35 # 35% max per sector
  KELLY_FRACTION = 0.25        # Quarter Kelly for safety
  ```

  ```python
  def kelly_criterion(win_prob: float, win_loss_ratio: float) -> float:
  ```
  - f* = (p * b - q) / b where p=win_prob, b=win_loss_ratio, q=1-p
  - Return max(0, f*) -- never negative (don't short in v1)

  ```python
  def optimizer(state: InvestmentState, config: RunnableConfig) -> dict:
  ```

  Implementation:
  1. Read from state:
     - ticker = state["asset_ticker"]
     - recommendation = state.get("judge_recommendation", "HOLD")
     - conviction = state.get("judge_conviction", 0.5)
     - consensus_dist = state.get("judge_consensus_distribution", {})
     - price_target = state.get("judge_price_target", 0)
     - guardian_sizing = state.get("guardian_sizing", 0.05)

  2. If recommendation is SELL or guardian_veto is True:
     - position_size = 0.0 (no allocation)

  3. Else compute Kelly sizing:
     - win_prob = conviction (Judge conviction as win probability proxy)
     - expected_gain = (price_target - current_price) / current_price if price_target > current_price
     - expected_loss = guardian margin of safety (downside estimate)
     - win_loss_ratio = expected_gain / expected_loss if expected_loss > 0 else 1.0
     - raw_kelly = kelly_criterion(win_prob, win_loss_ratio)
     - fractional_kelly = raw_kelly * KELLY_FRACTION
     - position_size = min(fractional_kelly, MAX_POSITION_SIZE, guardian_sizing * 1.5)

  4. Sector allocation (simplified for single-asset v1):
     - Determine sector from yfinance: yf.Ticker(ticker).info.get("sector", "Unknown")
     - Apply MAX_SECTOR_ALLOCATION cap
     - optimizer_sectors = {sector: position_size}

  5. Portfolio metrics (simplified):
     - expected_return = position_size * (price_target / current_price - 1) if price_target > 0
     - portfolio_risk = position_size * consensus_dist.get("std_dev", 20) / consensus_dist.get("mean", 100)
     - sharpe = expected_return / portfolio_risk if portfolio_risk > 0 else 0

  6. Invoke LLM (MODEL_FLASH) for narrative:
     - System: "You are a portfolio optimizer. Explain the position sizing rationale using Kelly Criterion and risk constraints."
     - Include: raw Kelly, fractional Kelly, caps applied, final size, risk metrics.

  7. Return:
     ```python
     return {
         "optimizer_portfolio": {ticker: position_size},
         "optimizer_sectors": {sector: position_size},
         "optimizer_rebalancing": [],  # v1: no existing positions
         "optimizer_metrics": {
             "raw_kelly": raw_kelly,
             "fractional_kelly": fractional_kelly,
             "position_size": position_size,
             "expected_return": expected_return,
             "portfolio_risk": portfolio_risk,
             "sharpe": sharpe,
             "max_drawdown_estimate": -portfolio_risk * 2,
         },
         "optimizer_narrative": narrative,
     }
     ```
