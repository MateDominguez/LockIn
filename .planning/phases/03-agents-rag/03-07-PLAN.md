     - Connect to DATABASE_URL from Settings
     - Create tables:
       ```sql
       CREATE EXTENSION IF NOT EXISTS vector;

       CREATE TABLE IF NOT EXISTS documents (
           id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
           source_type TEXT NOT NULL,  -- 'pdf', '10k', 'transcript'
           source_id TEXT NOT NULL,    -- filename, CIK, ticker+quarter
           title TEXT,
           metadata JSONB DEFAULT '{}',
           created_at TIMESTAMPTZ DEFAULT now(),
           UNIQUE(source_type, source_id)
       );

       CREATE TABLE IF NOT EXISTS chunks (
           id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
           document_id UUID REFERENCES documents(id),
           content TEXT NOT NULL,
           chunk_index INT NOT NULL,
           is_parent BOOLEAN DEFAULT false,
           parent_id UUID REFERENCES chunks(id),
           metadata JSONB DEFAULT '{}',
           created_at TIMESTAMPTZ DEFAULT now()
       );

       CREATE TABLE IF NOT EXISTS embeddings (
           id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
           chunk_id UUID REFERENCES chunks(id),
           embedding vector(768),  -- text-embedding-004 dimension
           metadata JSONB DEFAULT '{}',
           created_at TIMESTAMPTZ DEFAULT now()
       );

       CREATE INDEX IF NOT EXISTS idx_embeddings_vector ON embeddings USING ivfflat (embedding vector_cosine_ops) WITH (lists = 100);
       ```
     - Print success message

  2. Create src/lockin/rag/ingestion.py:
     ```python
     def ingest_pdf(file_path: str, source_id: str | None = None, metadata: dict | None = None) -> str:
     ```
     - Use pypdf to extract text from PDF
     - Split into parent chunks (1500 chars, 200 overlap) and child chunks (500 chars, 50 overlap) using RecursiveCharacterTextSplitter from langchain_text_splitters
     - Store document record in documents table
     - Store parent and child chunks in chunks table
     - Embed child chunks using GoogleGenerativeAIEmbeddings(model="models/text-embedding-004")
     - Store embeddings in embeddings table
     - Return document_id

     ```python
     def ingest_10k(ticker: str, years: int = 3) -> list[str]:
     ```
     - Use edgar library: `company = edgar.Company(ticker)`, `filings = company.get_filings(form="10-K").latest(years)`
     - For each filing, extract text content
     - Call ingest logic (same chunking/embedding as PDF)
     - Return list of document_ids

     ```python
     def ingest_transcript(ticker: str, quarter: int, year: int, transcript_text: str) -> str:
     ```
     - Store earnings call transcript with metadata (ticker, quarter, year)
     - Same chunking/embedding pipeline
     - Return document_id

  Important: All ingestion functions should be idempotent (UPSERT on source_type+source_id).
